## Project Description

This repository contains a multi-module Laravel-based application designed for job management and backoffice operations. The project is structured into several main components:

- **job-app/**: The main job application platform, built with Laravel, providing core features for job seekers and employers.
- **job-backoffice/**: The administrative backoffice for managing users, jobs, and platform settings, also built with Laravel.
- **job-shared/**: Shared libraries and resources used across both the job application and backoffice modules.

Each module is a standalone Laravel application, allowing for modular development and deployment. The project leverages Laravel's expressive syntax, robust routing, Eloquent ORM, and modern PHP best practices.

### Features
- Modular monorepo structure for scalable development
- User authentication and authorization
- Job posting and management
- Administrative dashboard
- Shared codebase for common logic

### Getting Started
1. Clone the repository.
2. Install dependencies for each module using Composer and NPM.
3. Configure environment variables for each application.
4. Run migrations and seeders as needed.
5. Start the development servers.

For detailed setup instructions, refer to the README.md files inside each module directory.
